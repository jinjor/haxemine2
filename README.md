Haxemine
======================
A simple editor for Haxe and Jasmine

install

    npm install -g path/to/haxemine


haxemine.json

    {
      "port": 8765,
      "hxml": [
        {
          "path": "compile.hxml"
        },{
          "path": "build.hxml"
        }
      ]
    }
    
run

    haxemine
